## 如何使用
+ 下载本项目
+ 先登陆微博，切换到发微博的页面
+ chrome 浏览器 菜单 -> 更多工具 -> 扩展程序 -> 加载已解压的扩展程序 -> 选择本项目文件夹
+ 回到微博页面，刷新，插件自动运行
+ 点击按钮右上角的面板按钮，运行程序
